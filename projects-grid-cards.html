JustBuild - Web App and Mobile App

“JustBuild” is my second-year software project done in a collaboration with the Al Tayer Group. This provides construction project scheduling, cost management in construction projects, and other features that allow working more efficiently and productively. Our goal was to give the most required features for managing construction-based projects with modern technology. Automated Portable Pedestrian Cross Light System

The traffic control system assists pedestrian detection with the help of ultrasonics and automates the traffic lights and gates. An Atmega 32A with integrated Xbee modules is used as a microcontroller with support for wireless communication. Its main contribution is to develop a portable, wireless, economical, and automated traffic control system that automatically monitors pedestrians, and automatically controls traffic lights for safe and efficient service.
